# Student Management System (Python)

## Description
A simple command-line Student Management System built with Python.  
It supports authentication, role-based access, course management, and GPA calculation.

## Features
- Student registration
- Login system
- Logout system
- Admin dashboard access
- Role-based access control
- Course score management
- GPA calculation
- JSON file storage
